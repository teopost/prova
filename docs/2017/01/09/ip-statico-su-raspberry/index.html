IP Statico su Raspberry

09-01-2017 · 116 words · 1 minute read
linux
raspberry

Se si vuole utilizzare una raspberry come un server, la prima cosa da fare è evitare che l’indirizzo IP cambi ad ogni reboot. Insomma va disabilitato il dhcp. Per farlo agire come segue:

Editare il file /etc/network/interfaces

sudo vi /etc/network/interfaces

Commentare iface eth0 inet manual e aggiungere in fondo al file i dati dell’ip:

# iface eth0 inet manual
auto eth0
iface eth0 inet static
address 192.168.10.32
gateway 192.168.10.254
netmask 255.255.255.0

Per motivi ignoti il vecchio ip continua a rimanere assegnato se non si disinstalla il dhcp. Lo si può verificare digitando:

ip addr show

Si può risolvere il problema disinstallando il relativo pacchetto

apt-get purge dhcpcd5

Fare un reboot e ricontrollare con:

ip addr show
