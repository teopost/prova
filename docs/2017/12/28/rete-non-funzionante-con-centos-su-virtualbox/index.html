Rete non funzionante con CentOS su VirtualBox

28-12-2017 · 474 words · 3 minute read
linux
virtual-machines

Ho scaricato da osboxes.org una VM con dentro CentoOS (sono pigro e se posso uso cose già fatte). Ho avviato la macchina virtuale e ho provato a navigare su internet. Non funzionava. Eseguendo il comando ifconfig non appariva la scheda di rete. Eppure sembrava tutto configurato bene. Ecco come ho soluzionato, come direbbe il mio amico coniglio.

Soluzione

  • Aprire il terminale
  • Diventare root (su - root)
  • Eseguire dhclient –v

Eseguire il comando allo startup

Purtroppo se spegnete la macchina virtuale vi accorgerete che questa soluzione (se così la vogliamo chiamare) non è persistente. Se si vuole rendere definitiva seguire i seguenti passi:

  • Andare in /etc/init.d
  • Creare un file chiamato net-autostart
  • Incollarci dentro sta roba:
#!/bin/bash
# Fix for "No Internet Connection"
#
### BEGIN INIT INFO
# Default-Start: 2 3 4 5
# Default-Stop: 0 1 6
### END INIT INFO
dhclient -v
  1. Salvare il file
  2. Cambiare i permessi del file (chmod 755 net-autostart)
  3. Aggiungere l’esecuzione automatica dello script (chkconfig --add net-autostart)
  4. Fare un bel reboot
